Onboarding Flow (10-Step Wizard)

SCREEN-05: Welcome

Built /app/onboarding Step 0
First screen after signup — welcome and motivate
  • Animated progress bar (0/10)
  • Welcome message and app intro
  • "Let's get started" CTA
Note: Should feel premium and motivating. Sets the tone for the whole app.

SCREEN-06: Name

Built Step 1
Collect name for personalization
  • Text input for display name
Note: Optional — enables "Hey, {name}" on dashboard.

SCREEN-07: Date of Birth

Built Step 2
Collect DOB for calorie calculation (BMR)
  • Date picker input

SCREEN-08: Biological Sex

Built Step 3
Collect biological sex for BMR accuracy
  • 3-button grid: Male, Female, Other

SCREEN-09: Measurements

Built Step 4
Collect height and weight
  • Height input (cm, min 50)
  • Weight input (kg, min 20)
Note: Keep fast — minimize friction. Units toggle (lbs/kg) planned.

SCREEN-10: Activity Level

Built Step 5
Assess daily activity for calorie multiplier
  • 5-option list: Sedentary, Lightly Active, Moderately Active, Active, Very Active

SCREEN-11: Goal Selection

Built Step 6
User selects primary fitness goal
  • 3 goal cards with emoji: Lose Weight, Maintain, Gain Weight
Note: Determines calorie adjustment (-500 for lose, +300 for gain) and macro splits.

SCREEN-12: Dietary Preference

Built Step 7
Capture diet type for recipe/meal suggestions
  • 6-button grid (Standard, Vegetarian, Vegan, Keto, Paleo, etc.)
Note: Optional. Informs AI Coach and recipe recommendations.

SCREEN-13: Food Allergies

Built Step 8
Capture allergens for safe recommendations
  • 8-option multi-select grid of common allergens
Note: Critical for AI Coach safety — never suggest flagged allergens.

SCREEN-14: Review & Confirm

Built Step 9
Show calculated targets — the payoff moment
  • Summary card with all collected info
  • Large calorie target display
  • 3-column macro grid: Protein, Carbs, Fat with color coding
  • Goal, Activity, Diet, Allergies summary
  • "Complete Setup" button
Note: This is the payoff — user sees their personalized plan. Make it feel rewarding.
Core App Screens

SCREEN-15: Home Dashboard

Built /app/dashboard Home Tab
Daily overview and command center — the most important screen
Components
  • Personalized greeting: "Hey, {name}"
  • Onboarding banner (if profile incomplete)
  • Calorie Ring (animated SVG)
  • Circular progress — remaining calories in center
  • Green if under target, red if over
  • "X / Y kcal" badge below
  • Macro Bars (3 animated rows)
  • Protein (blue), Carbs (amber), Fat (red)
  • Color dot + label + "X / Y g" + progress bar
  • Quick Actions (2x2 grid)
  • Log Meal, Snap Photo, Ask Coach, Scan Barcode
  • AI Coach Insight Card
  • Dynamic coaching message based on daily progress
  • "Chat with coach" link
Note: The "command center." Everything a user needs at a glance. Smart coaching messages adapt to progress.

SCREEN-16: Meal Log

Built /app/log Log Tab
Log meals and view daily food intake — speed is king
Components
  • Mode tabs: Search | Photo | Scan
  • Search Mode (default)
  • Search input with magnifying glass
  • "Recent Foods" list with name, serving, calories, protein
  • Plus icon to quick-add
  • Photo Mode
  • Camera icon card — "Snap your meal" + "Open Camera"
  • Scan Mode
  • Barcode icon card — "Scan a barcode" + "Open Scanner"
  • Meal Type Selector: Breakfast | Lunch | Dinner | Snack
Note: Must be FAST — minimal taps to log a food item. The daily driver of the app.

SCREEN-17: Food Search & Add

Partial /app/log (search mode)
Search food database and add items to a meal
  • Search bar with recent/favorites tabs (planned)
  • Food results with calorie/macro preview
  • Serving size adjuster (planned)
  • Add to meal button
  • Barcode scan shortcut
Note: Currently embedded in Meal Log search mode. May need its own dedicated view as the food database grows.

SCREEN-18: AI Coach Chat

Scaffold /app/coach Coach Tab
Conversational AI for nutrition/fitness guidance — core differentiator
Planned Components
  • Chat message bubbles
  • Suggested quick-reply chips
  • Context cards (user's current stats shown inline)
  • Typing indicator
  • Clear chat, export, adjust AI persona options
Note: Should feel like texting a knowledgeable friend. Currently shows empty state with "Start a conversation" prompt.

SCREEN-19: Recipes Library

Scaffold /app/recipes
Browse and save recipes matching user goals
Planned Components
  • Recipe cards with photo, title, macros, prep time
  • Filter by meal type, diet, prep time, macro goals
  • Search bar
  • "Saved" / "Favorites" tab
  • AI-generated suggestions based on remaining macros
Note: "What can I make with my remaining macros?" is the killer feature. Desktop sidebar only (not in mobile nav).

SCREEN-20: Progress & Analytics

Scaffold /app/progress Progress Tab
Visualize long-term trends — motivational
Planned Components
  • Weight chart (line graph over time)
  • Calorie adherence heatmap (calendar view)
  • Macro breakdown trends
  • Streak counter / consistency score
  • Milestone badges
Note: Should make users feel good about their progress. Unlocks after 3 days of logging.

SCREEN-21: Profile & Settings

Built /app/profile Profile Tab
User profile, preferences, and app settings
  • Avatar + display name (inline editable) + email
  • Daily Targets: Calories, Protein, Carbs, Fat (color-coded cards)
  • User Info: Goal, Activity, Weight, Height
  • "Recalculate nutrition targets" link
  • Sign out with confirmation flow
Note: Keep clean — don't overcrowd. Inline editing, no modals.
